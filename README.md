# Aqua Tick

URL:https://localhost:5173

揺れ動く水をイメージした時計です。

配信などでご利用いただけます。

### 使い方

1. 「ソースを追加」→「ブラウザ」を選択
2. URL に「https://localhost:5173」を入力
3. 幅を **600px**、高さを **600px** に設定
4. 「OK」を押すと時計が表示されます
